## 調理方法
1. タッパーに具材を入れる
2. タッパーの空き具合を見て、野菜の入れる量を調整する
3. タッパーに蓋をする
4. タッパーを電子レンジに入れて、電子レンジで加熱して蒸す
5. 具材の量により、調理時間は5分から10分
